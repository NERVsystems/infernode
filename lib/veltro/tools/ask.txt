ask - Prompt user for input

Usage:
  ask <question>                      Free-form input
  ask -c <choice1,choice2> <question> Multiple choice

Arguments:
  question - The question to ask
  -c       - Comma-separated list of valid choices

Examples:
  ask "What is the target directory?"
  ask -c "yes,no" "Do you want to continue?"
  ask -c "small,medium,large" "What size?"

For multiple choice, user can enter number (1, 2, 3) or text.
Returns the user's response as a string.
