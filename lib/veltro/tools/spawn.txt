spawn - Parallel subagents with isolated namespaces

  Spawn -- tools=<tools> :: <task>
  Spawn -- tools=t1 :: task A -- tools=t2 :: task B   (parallel)

  tools=     (required) Comma-separated: "read,list,grep"
  paths=     Host paths: "/n/local/Users/me/src"
  agenttype= explore | plan | task | default
  model=     haiku (default) | sonnet | opus

Results: === Subagent N: task === blocks. Task must not contain ' -- '.

  Spawn -- tools=read,list :: summarize /appl/veltro
