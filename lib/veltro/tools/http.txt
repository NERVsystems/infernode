http - HTTP client

Usage:
  http GET <url>              GET request
  http POST <url> <body>      POST request
  http PUT <url> <body>       PUT request
  http DELETE <url>           DELETE request
  http HEAD <url>             Headers only

Arguments:
  url  - Full URL (http:// or https://)
  body - Request body for POST/PUT (JSON)

Examples:
  http GET http://example.com/api
  http POST http://localhost:8080/data '{"key": "value"}'

Notes:
  - Requires /net access (trusted agents only)
  - HTTPS requires /net/ssl (may not be available)
  - Returns response body or error with HTTP status
