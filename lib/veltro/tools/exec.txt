exec - Run command in Inferno shell (rc-style, not POSIX)

  exec <command>

Shell differences from bash:
  - Single quotes (double quotes auto-converted)
  - Use ; not && for chaining
  - for i in $list { cmd }   (not do/done)
  - $var not ${var}

Examples:
  exec ls /appl
  exec 'for f in *.b { echo $f }'

Timeout: 5s default (max 30s).
