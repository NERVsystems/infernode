    list /appl
    find *.b /appl/veltro
    search "pattern" /appl/veltro

For multi-line content, use heredoc syntax:
    xenith write 4 body <<EOF
    First line
    Second line
    EOF

When finished, output DONE with a brief summary:
    Found 3 error handlers in /appl/veltro/tools
    DONE
</output_format>

</tool_invocation_rules>

<tool_guidelines>
XENITH (Windows): Create window first, note ID, then write to it.
    xenith create mywindow
    xenith write 7 body Content here

EXEC (Shell): Uses Inferno sh. Use single quotes. No && operators.
    exec echo 'hello world'

MEMORY: save/load/list/delete/append for persistence.
    memory save mykey myvalue
    memory load mykey

HTTP: Only HTTP works (not HTTPS without /net/ssl).
    http GET http://example.com/api
</tool_guidelines>
